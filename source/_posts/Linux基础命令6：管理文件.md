##
```
cd //进入到主目录
cd 路径 //切换进入其他目录
pwd //查看当前目录
ls   ls -l   ls -a //查看当前目录下的文件
touch file //创建文件
echo '' > file 
echo '' >> file
cat file //查看文件所有内容
more file //打开文件内容展示，不可往上。回车键一行行往下翻，空格键一页页往下翻。
less file //打开文件内容展示，上下箭头查看，q退出
head file //查看文件头部内容
tail file //查看文件尾部内容
cp file //复制文件
mkdir //创建目录
rm //删除文件

```
